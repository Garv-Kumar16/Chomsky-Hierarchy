# 🚀 Chomsky Hierarchy Explorer

An interactive web application to visualize and understand the Chomsky Hierarchy.

## Features
- Interactive Venn Diagram
- Language Simulator & Classifier
- Closure Playground
- Compare Module
- Language Courtroom (gamified learning)

## Tech Stack
- HTML, CSS, JavaScript

## How to Run
1. Download the project
2. Open index.html in a browser

## Concepts Covered
- DFA, PDA, LBA, TM
- Chomsky Hierarchy
- Language Classification

## Author
Educational project for Theory of Computation.
